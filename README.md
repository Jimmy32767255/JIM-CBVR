# JIM-CBVR

## 一个自由，开放的VR串流解决方案

## 当前功能：

### Windows服务端：
 - [x] WPF配置界面(含JSON序列化)
 - [x] SteamVR连接核心模块
 - [x] 智能端口管理(自动冲突检测)
 - [x] 画面矫正参数配置界面
 - [x] 双模式网络服务(USB/WiFi)

### Android客户端：
 - [x] OpenGL ES 3.0渲染管线
 - [x] WebSocket双通道连接
 - [x] 动态畸变参数更新
 - [x] 心跳检测机制

## 为什么创建它？

### 因为RiftCat VRidge收费，Trinus VR又不好用，而且不美观，所以我决定自己做一个！
